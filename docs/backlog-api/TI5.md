# TI-5: Governança e ação

**Duração**: ~3 semanas  
**Prioridade**: 🟡 P1  
**Trilha**: Inteligência Territorial  
**Depende de**: [TI4](./TI4.md) · jornadas governance/events existentes  
**Ancora fases**: [FASE14](./FASE14.md), [FASE24](./FASE24.md) (ações territoriais)  
**Status**: ⏳ Pendente

---

## Objetivo

Alerta vira resposta coletiva: mutirão/evento ou votação com origem no alerta; papel de validador institucional; protocolos da memória.

---

## Entregas

- Evento/votação com `originAlertId`
- Papel validador institucional (selo auditado, por território)
- Protocolos reutilizáveis da memória (ex.: limpar bueiros em outubro)
- Spec TI-501
- Cenário Socorro passos 11–12

---

## Critérios de aceite

- [ ] Mutirão/votação criáveis a partir do alerta com rastreio de origem
- [ ] Selo institucional auditado (ator + competência)
- [ ] Parcerias atrasadas não bloqueiam: papel entra por território, sem big-bang
- [ ] Cenário Socorro 11–12
