A1

Sobre este adendo

A spec-mãe descreve a camada de experiência território-primeiro. Este adendo acrescenta a camada econômica que a torna sustentável e geradora de renda para implementadores — sem fechar o código nem cobrar do morador. Ele não substitui nenhuma seção: estende as jornadas Mercado & checkout, Minha loja, Carteira territorial e Governança & votação, e adiciona domínios, endpoints e telas.

Lógica de billing & split

Ordem de cálculo no servidor. O cliente apenas exibe o resultado; nunca calcula nem aplica taxa.

{{ b.n }} {{ b.text }}
Exemplo trabalhado · venda de R$ 60,00
grossAmount = R$ 60,00
feeRate (mercado) = 2,9%  →  feeAmount = R$ 1,74
vendedor recebe = R$ 58,26
split do fee (R$ 1,74):
· implementador 45% = R$ 0,78
· fundo do território 25% = R$ 0,44
· plataforma 30% = R$ 0,52
