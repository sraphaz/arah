G7

Base open-source

Não reinventar: aproveitar o ecossistema OSS de agentes — adotar, adaptar ou inspirar. Coerente com o open-core e com a soberania da Arah (modelos open-weight self-hostáveis = soberania de IA).

Projeto OSSPara quê na ArahComo usar
{{ o.name }} {{ o.use }} {{ o.mode }}
vpn_lockModelos open-weight, self-hostáveis

Para código: Qwen2.5-Coder, DeepSeek-Coder, Codestral, Llama 3.x, StarCoder2 — servidos por Ollama ou vLLM. Isso permite rodar agentes sem que o código/dados do território saiam da infra — o mesmo princípio da instância soberana, agora para a IA.

Estratégia híbrida: modelo de fronteira (gerenciado) para tarefas difíceis; open-weight local para o volume e para territórios soberanos.
